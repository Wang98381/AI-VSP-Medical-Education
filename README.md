# AI-VSP-Medical-Education
A Virtual Standardized Patient system based on DeepSeek for medical history-taking training. (Collaboration Welcome)
# AI虚拟标准化患者(VSP)医学教学系统

**发起人**：上海交通大学医学院附属医院肝胆外科主任医师 | 医学教育项目

## 🎯 项目概述
我们正在构建一个基于DeepSeek大模型的虚拟标准化患者交互系统，用于医学生的病史采集与临床思维训练。本项目旨在探索AI在医学教育中的创新应用。

## 🛠 技术栈与开发需求
- **后端**：Python (FastAPI/Flask)
- **前端**：React/Vue (基础界面)
- **核心AI**：DeepSeek API集成、Prompt工程、对话状态管理
- **部署**：Docker，支持私有化部署
